# 請假時間估算系統

輸入今日打卡時間與預計離開時間，自動計算需要填寫的假單張數、時段與時數。

## 功能

- **遲到補假**：09:31 後打卡，自動以 09:30 為基準計算補假時段
- **彈性上班調整**：09:01–09:30 到班且同日提早離開，自動往前調整有效上班時間
- **提早離開 / 早退假**：計算下午或上午離開所需的假單時段
- **午休自動處理**：跨越 12:00–13:00 一律合併為一張假單，無需手動拆分
- **最低時數進位**：假單最短 30 分鐘，超過後以 5 分鐘為單位無條件進位

## 請假規則

| 情境 | 說明 |
|------|------|
| 08:00–09:30 到班 | 依實際到班時間決定正常下班時間（在公司滿 9 小時） |
| 09:31 後到班 | 一律須填**遲到補假單**（以 09:30 為基準起算） |
| 09:01–09:30 到班 + 提早離開 | 須往前調整上班時間（一張晨間假單），18:01–18:30 不可作為請假時段 |
| 中午離開（12:00–13:00） | 等同 12:00 離開，下午假從 13:00 起算 |
| 09:31 後打卡 + 同日提早離開 | 需填兩張假單（遲到補假 + 早退假） |

## 使用方式

直接用瀏覽器開啟 `index.html`，無需安裝任何套件或伺服器。

```
open index.html
```

1. 選擇**今日上班打卡時間**
2. 選擇**預計請假離開時間**
3. 點擊「計算假單」
4. 查看輸出的假單時段與總時數
